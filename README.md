
#Field Guide

### Description
Field Guide allows users to browse or create custom guides to their city.

### Link to Heroku
<http://field-guide.herokuapp.com/>

### API's used
* Leaflet/MapBox
* Foursquare

### Features
* Leaflet Angular Directive
* Angular UI Bootstrap (typeahead search)
